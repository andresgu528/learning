HTML

Here I will list the things that I want to memorize from HTML, following the structure presented in w3schools.com/html.

Introduction

<!DOCTYPE html>

The <!DOCTYPE> declaration represents the document type. For HTML5 documents it must be as stated.

<html>content</html>

Root element of an HTML page. It's where the HTML document itself begins.

<head>content</head>

Contains meta information about the page.

<title>content</title>

Specifies content as the title for the page, shown either in the title bar or in the page's tab.

<body>content</body>

Defines the document's body, and is a container for all the visible contents.

<h1>content</h1>

Defines the most important heading.

<p>content</p>

Defines a paragraph.

<br>

Inserts a line break.

Basic

<h1>content</h1> to <h6>content</h6>

Define headings from most important to least important.

<a href="dest_url">content</a>

Defines an hyperlink to dest_url that is activated when clicking in content.

<img src="img_url" alt="alt_text" width="width" height="height">

Embeds the image specified by img_url, with alt, width and height attributes, that set the alternative text, width, and height, respectively.

Attributes

<p style="color:red;">content</p>

The style attribute is used to add styles to an element. In the code shown, it sets the font color of content to red.

<html lang="en-US">content</html>

The lang attribute declares the language of the element. Should always be included in the <html> tag to declare the language of the web page. It can also optionally specify the country. In the code shown, it specifies the language as English and the country as United States.

<p title="tooltip">content</p>

The title attribute defines extra information about an element. In the code shown, it displays tooltip when the mouse is hovered over content.

Comments

<!-- comment -->

Adds a comment to the HTML source, that is not displayed on the browser.

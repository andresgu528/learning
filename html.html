HTML

Here I will list the things that I want to memorize from HTML, following the structure presented in w3schools.com/html.

Introduction

<!DOCTYPE html>

The <!DOCTYPE> declaration represents the document type. For HTML5 documents it must be as stated.

<html>content</html>

Root element of an HTML page. It's where the HTML document itself begins.

<head>content</head>

Contains meta information about the page.

<title>content</title>

Specifies content as the title for the page, shown either in the title bar or in the page's tab.

<body>content</body>

Defines the document's body, and is a container for all the visible contents.

<h1>content</h1>

Defines the most important heading.

<p>content</p>

Defines a paragraph.

<br>

Inserts a line break. It can be used for adding a new line without starting a new paragraph.

Basic

<h1>content</h1> to <h6>content</h6>

Define headings from most important to least important.

<a href="dest_url">content</a>

Defines an hyperlink to dest_url that is activated when clicking in content.

<img src="img_url" alt="alt_text" width="width" height="height">

Embeds the image specified by img_url, with alt, width and height attributes, that set the alternative text, width, and height, respectively.

Attributes

<p style="color:color_value;">content</p>

The style attribute is used to add styles to an element. In the code shown, it sets the font color of content to color_value. One of the values that it can take is red.

<html lang="en-US">content</html>

The lang attribute declares the language of the element. Should always be included in the <html> tag to declare the language of the web page. It can also optionally specify the country. In the code shown, it specifies the language as English and the country as United States.

<p title="tooltip">content</p>

The title attribute defines extra information about an element. In the code shown, it displays tooltip when the mouse is hovered over content.

Comments

<!-- comment -->

Adds a comment to the HTML source, that is not displayed on the browser.

Headings

<h1 style="font-size:valuepx;">content</h1>

The font-size CSS property for the style attribute specifies the text size of content, in this case setting it to value pixels, where value is a number.

Paragraphs

<hr>

Defines a thematic break, and is usually displayed as an horizontal rule. It is used to separate content or define a change in the page.

<pre>content</pre>

Defines a preformatted text, preserving spaces and line breaks.

Styles

<p style="background-color:color_value;">content</p>

The CSS background-color property defines the background color of an element as color_value. Some values that it can take are powderblue, tomato and blue.

<h1 style="font-family:font_value;">content</h1>

The CSS font-family property defines the font to be used for an element as font_value. Some values that it can take are verdana and courier.

<p style="font-size:value%;">content</p>

In this case the font-size CSS property specifies the text size of content to be value% of the default size, where value is a number.

<p style="text-align:align_value;">content</p>

The CSS text-align property defines the horizontal text alignment of an element as align_value. One value that it can take is center.

Formatting

<b>content</b>

Defines bold text, without any extra importance.

<strong>content</strong>

Defines text with strong importance.

<i>content</i>

Defines a part of text in an alternative voice or mood. Often used to indicate technical terms, phrases from another languages, etc.

<em>content</em>

Defines emphasized text.

<small>content</small>

Defines smaller text.

<mark>content</mark>

Defines text that should be marked or highlighted.

<del>content</del>

Defines text that has been deleted from a document.

<ins>content</ins>

Defines text that has been inserted into a document.

<sub>content</sub>

Defines subscript text, that appears half a character below the normal line.

<sup>content</sup>

Defines superscript text, that appears half a character above the normal line.

Quotations

<blockquote cite="quote_source">content</blockquote>

Defines a section that is quoted from the source quote_source.

<q>content</q>

Defines a short quotation.

<abbr title="abbr_description">content</abbr>

The HTML abbr tag defines an abbreviation. The title attribute can be used to specify and show its description abbr_description.

<address>content</address>

Defines the contact information for the writer or owner of the document or article.

<cite>content</cite>

Defines the title of a creative work.

<bdo dir="rtl">content</bdo>

Used to override the direction of content. BDO stands for bi-directional override.

CSS

<style>css_content</style>

The <style> element defines the style for a single HTML page. Used inside the <head> section. The structure of css_content is as follows:

element_1 {property_11: value_11; ... ; property_1j: value_1j; ... ; property_1n: value_1n;}
...
element_i {property_i1: value_i1; ... ; property_ij: value_ij; ... ; property_in: value_in;}
...
element_m {property_m1: value_m1; ... ; property_mj: value_mj; ... ; property_mn: value_mn;}

<link rel="stylesheet" href="css_file_location">

Used inside the <head> section to implement the CSS styles defined in the file located at css_file_location. Is common practice for this file to have the .css file extension.

<style>
p {border: numpx solid color_value>;}
</style>

The CSS border property defines a border around an HTML element. In the code above, it specifies for the <p> element a num pixels wide, color_value colored, solid border.

<style>
p {padding: numpx;}
</style>

The CSS padding property defines a padding between the content of the element and the border. In the code above, it specifies for the <p> element a padding of num pixels.

<style>
p {margin: numpx;}
</style>

The CSS margin property defines a margin outside the border. In the code above, it specifies for the <p> element a margin of num pixels.

HTML

Here I will list the things that I want to memorize from HTML, following the structure presented in w3schools.com/html.

Introduction

<!DOCTYPE html>

The <!DOCTYPE> declaration represents the document type. For HTML5 documents it must be as stated.

<html></html>

Root element of an HTML page. It's where the HTML document itself begins.

<head></head>

Contains meta information about the page.

<title></title>

Specifies a title for the page, shown either in the title bar or in the page's tab.

<body></body>

Defines the document's body, and is a container for all the visible contents.

<h1></h1>

Defines the most important heading.

<p></p>

Defines a paragraph.

<br>

Inserts a line break.

Basic

<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>

Define headings together with h1. From most important to least important.

<a href="link_destination">element_displayed</a>

Defines a link to link_destination that is activated when clicking in element_displayed.

<img src="source_file" alt="alternative_text" width="width>" height="height>">

Inserts the image specified by source file, with alt, width and height attributes.
